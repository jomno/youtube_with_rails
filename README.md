# rails youtube upload

## setup Gemfile
```
gem 'yt', '~> 0.13.7'
gem 'bootstrap-sass', '~> 3.3.0.1'
gem 'autoprefixer-rails'
gem 'rails-observers'
gem 'devise'
gem 'omniauth'
gem "omniauth-google-oauth2"  
gem "figaro"
gem 'rails_db'
gem 'active_type', '0.3.1'
```
